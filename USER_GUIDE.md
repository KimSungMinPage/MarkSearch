# 사용자 안내서

1. 제공받은 ZIP 파일을 원하는 폴더에 압축 해제합니다.
2. `GmpCompanyCollector.exe`를 더블클릭합니다.
3. Windows 보안 경고가 나오면 배포자를 확인한 뒤 신뢰할 수 있을 때만 실행합니다.
4. `API 설정` 탭에서 공공데이터포털 서비스키를 입력합니다.
5. 키 형식은 기본값 `자동 감지`를 권장합니다. Encoding 키는 `%2B`, `%2F`, `%3D`가 포함될 수 있습니다.
6. `API 연결 테스트`를 눌러 전체 건수와 응답 시간을 확인합니다.
7. `데이터 수집` 탭에서 업체 목록만, 제조소 상세정보만, 또는 둘 다를 선택합니다.
8. 필요하면 업종, 업체명, 허가일자, 허가번호 조건을 입력하고 `수집 시작`을 누릅니다.
9. 수집 결과 탭에서 `Excel 저장`을 눌러 `.xlsx` 파일을 저장합니다.
10. `CSV 저장`을 누르면 업체 목록과 제조소 상세 CSV가 `utf-8-sig`로 저장됩니다.
11. `로그` 탭에서 오류와 진행 내역을 확인할 수 있습니다.
12. 등록되지 않은 서비스키, 활용기간 만료, 요청 한도 초과 오류는 공공데이터포털 마이페이지에서 확인하십시오.
